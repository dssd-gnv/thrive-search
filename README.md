# Thrive Literature Review: Searchable Database

A responsive web application for searching and filtering the literature review for Thrive Central Florida.

Thrive Central Florida is a collaboration with the Central Florida Foundation that seeks to bring nonprofits, policymakers, business leaders, educators, community members, and experts together to make real change.

Florida Community Innovation's policy team reviewed hundreds of in-depth policy papers, case studies, and documents to summarize the best tactics with which to equip our changemaking partners across Central Florida.

Data Science for Sustainable Development's team built this searchable database to make it easy for anyone to find the information they need to create positive change in Central Florida.

## Getting Started

### Prerequisites

- Node.js 16.x or higher
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd thrive-search
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

### Building for Production

To build the application for production:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
